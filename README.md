# Cosecha Citrus

Aplicación web para llevar el control de asistencia y producción de cosecha.

## Ejecutar localmente

**Prerequisitos:** Node.js

1. Instalar dependencias:
   `npm install`
2. Iniciar la aplicación:
   `npm run dev`
3. Abrir en el navegador:
   `http://localhost:3000`

## Qué hace esta app

- Controla asistencia por cuadrilla y día
- Registra bines/bolsones de producción
- Administra colectivos y capataces
- Muestra métricas y gráficos en el dashboard
- Exporta los datos a Excel
